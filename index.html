Welcome to FSMDesigner; a web-based tool for Finite State Machine entry. Capturing a Finite State Machine is easy! Just use the following controls:
  • Double click any blank space to create a new state.
  • Single click any existing state to edit its name.
  • Double click any existing state to edit its output or outputs.
  • Click any existing state or transition then press Delete to delete it.
  • Shift + drag to add a new state transition.
Relax, and enjoy the FSM entry- FSMDesigner has your back!
If you're using a modern browser, your work is constantly autosaved- so you're safe even if your browser crashes, or you accidentally navigate away.
